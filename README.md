# 2048

2048 game built with React and Typescript.

## Installation and usage

```shell
#install dependencies
pnpm i
```

```shell
#run in dev mode
pnpm start 
```

```shell
#build prod version
pnpm build
```

```shell
#lint code
pnpm lint
```

```shell
#lint code and fix code style
pnpm lint:fix
```

```shell
#format code
pnpm format
```
